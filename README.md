Developed MeetNote AI, an accessibility-first intelligence hub that transforms meetings into structured, interactive insights for neurodivergent and deaf users.

Engineered a Multi-Model AI Pipeline using OpenAI Whisper for transcription and Google Gemini 1.5 Flash for deep semantic analysis and per-segment sentiment tracking.

Implemented Interactive Knowledge Graphs using Mermaid.js, enabling users to spatially visualize meeting logic and connections, reducing cognitive load.

Designed a "Vibe Meter" Sentiment Engine that provides real-time emotional context through visual emojis, serving as "visual hearing" for non-auditory users.

Built for Reliability by implementing an NLTK-based local NLP fallback system, ensuring service continuity and 100% uptime for critical summaries.
